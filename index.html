🔑 How to get your API Key (use a secure Access Key) ▾

Important: JSONbin gives you a Master Key (full account access) and lets you create scoped Access Keys (limited access). For a browser app like FinTrack, always use an Access Key — it's safer because it can only touch your bins, not your whole account.

Steps (done on the jsonbin.io website, not here):

  1. Go to jsonbin.io and log in (or sign up — it's free).
  2. Open the API Keys page from your account dashboard.
  3. Find the Access Keys section (below your Master Key).
  4. Click Create Access Key, name it “FinTrack”.
  5. Grant these permissions: Bins Create, Bins Read, Bins Update. Leave Delete off.
  6. Copy the generated Access Key and paste it in the field below.

⚠️ You can't edit a key's permissions after creating it — if you need to change them, delete the key and create a new one. Avoid pasting your Master Key here.

💡 How To Use FinTrack
📊 Welcome to FinTrack
FinTrack is a single-file personal finance dashboard that runs entirely in your browser. No account needed, no data sent to any server — your data stays on your device. Everything flows from one transaction log; all tabs update automatically.
Designed & built by Tushar Jain  ·  © All rights reserved
▶ Quick Start — 5 Steps
1
Set up Profile — Settings → Profile. Enter your name and 4-digit passcode.
2
Customise Categories — Settings → Transaction To. Rename SIP 1–4 to your fund names, add expense categories.
3
Add Banks & Cards — Settings → Transaction From. They appear in Bank and CC tabs automatically with auto-assigned colours.
4
Log Transactions — Transactions tab → + Add Transaction. Fill Date, Amount, From, To, Notes. All tabs refresh instantly.
5
Cloud Backup — Settings → Cloud Sync. Free account at jsonbin.io + API Key + Bin ID. Use ↓ Cloud in the top bar to sync across devices.
📋 Transactions — The Core
Add Transaction
Click + Add Transaction. Fill Date, Amount (₹), From (bank/card), To (category), Notes. All tabs update instantly.
From
Source of funds — bank account or credit card. Add in Settings → Transaction From.
To
Where money goes — expense, investment, loan EMI, or CC bill payment. Manage in Settings → Transaction To.
Filter
Click the Filter funnel button (top-right). Filter by Month, From, To, Search text, Min/Max amount. Active filters show as blue tags. Clear All resets.
Delete
Click ✕ on any row. A confirmation prompt prevents accidental deletion.
FY Selector
Yellow dropdown in the tab header. Changes FY for that tab only. Indian FY = April–March.
💳 Credit Card Tab
Add / Update Bill
Click + Add / Update Bill. Select card, enter statement amount and due date. Records what the bank billed you — separate from your actual payment.
Record Payment
Transactions → Add Transaction. From = bank, To = CC Bills → [Card]. Records the actual payment made.
Summary Cards
Total billed, paid, outstanding per card. Scroll right for more than 4 cards — existing cards keep their full size.
Spend Chart
Monthly spending per card. Filter pills let you isolate one card with a smooth animation.
Add New Card
Settings → Transaction To → CC Bills → + Add. Card appears here automatically with its own colour.
🏛 Bank Accounts Tab
Opening Balance
Your actual bank balance at FY start (or when you begin tracking). Starting point for running balance.
Running Balance
Opening Balance + credits (salary, income) − debits (expenses, investments, CC payments, loans).
Add New Bank
Settings → Transaction From → Bank Accounts → + Add. Appears automatically with its own colour.
📈 Investment Tab
Opening Balance
Total invested before you started tracking. Enter once as your historical cost basis. Update each April with actual portfolio value.
This FY
Auto-sum of investment transactions this FY. Just keep logging — it updates automatically.
Portfolio Value
Opening Balance + This FY = total invested cost (not market value). Update Opening Balance each April to track growth year-over-year.
Pie Toggle
This FY = only this year's investments. Portfolio = full picture (OB + This FY). Zero-value investments excluded from both views.
Lumpsum / Irregular
Log each investment on its actual date. Months with no investment show —. Annual totals are always accurate.
💸 Budget Planner
Annual Budget
Enter your planned spend for the full year per category (e.g. ₹60,000 for Groceries for the whole year).
Actual Spent
Auto-calculated from all transactions in that category this FY. Updates as you log transactions.
Status & Progress
✅ On Track = under budget. ⚠ Over = exceeded. Bar is green (under 80%), amber (80-100%), red (over budget).
📋 Loan Tracker
Log EMI
Transactions → Add Transaction. From = bank, To = Loans → [Loan Name]. Shows in Loan tab in the correct month.
Loan Month Override
Pay May EMI with June salary? Use Loan Month selector to attribute it to May for accurate tracking.
Interest Rate
Update if rate changes. Only affects future months, not months already paid.
⚙ Settings Tab
Profile
Name, initials, 4-digit passcode, auto-lock timer. Set this up first — your name appears on the lock screen.
Transaction To
All categories. Click section headers to expand. + Add to add items, ✕ to delete. Changes reflect everywhere instantly.
Transaction From
Banks and credit cards. Adding here makes them appear in Bank and CC tabs automatically.
Cloud Sync
JSONbin.io API Key + Bin ID. ↓ Cloud in the top bar loads your latest backup on any device.
Financial Year
Set default FY (Indian FY = April–March). Each tab also has its own FY selector for historical views.
Data Management
Export a full JSON backup anytime and restore it later with one click. Export CSV for viewing in Excel. Clear Data removes transactions only — categories and settings are preserved.
💾 Saving & Access
Local Save
Click 💾 Save (top bar). Stored in browser localStorage. Works fully offline.
Cloud Sync
JSONbin.io in Settings. ↓ Cloud loads your latest data on any device.
Privacy
Your data never leaves your device unless you set up Cloud Sync. No analytics, no tracking, no third-party services.
Cross-device
Download this HTML file, open in any browser. Pair with Cloud Sync to sync. No installation or subscription required.
💡 Pro Tips
Log Regularly
Weekly batch entry works fine. Consistency makes Dashboard and Budget more accurate.
Start of Each FY
Every April, update Opening Balances in Bank and Investment tabs with actual balances to reset the baseline.
New Sections
Adding a category section in Settings auto-creates a KPI card on Dashboard and Monthly Summary — no extra configuration.
CC Workflow
Add bill when statement arrives. Record payment as a transaction. Outstanding = Bill − Payment.
Auto Colours
Banks and cards get auto-colours by position: coral, blue, green, violet, amber, cyan, pink, lime, orange, indigo.
